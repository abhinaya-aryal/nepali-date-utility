# Nepali DatePicker Example

This directory contains example usage of the Nepali DatePicker component.

## Running the Example

```bash
# From the root directory
npm run dev
```

This will start the development server and you can see the example in action.

## Example Code

See `App.tsx` in the src directory for a complete example that showcases:

- Basic usage
- Format customization
- Locale switching
- Custom styling
- Min/Max date restrictions
- Position options
- Disabled state
